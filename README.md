## How to Run This Project on Your Computer


**STEP 1: Download the Project**<br>

<u>git clone https://github.com/Alakhx-dev/Restaurant-website1.git
</u><br>
<u>cd Restaurant-website1</u><br>

**STEP 2: Create Virtual Environment**<br>

<u>python -m venv .venv</u>

**STEP 3: Activate Virtual Environment**<br>

For Windows:<br>

<u>.venv\Scripts\activate</u>

For Mac / Linux:<br>

<u>source .venv/bin/activate</u>

**STEP 4: Install Required Packages**<br>

<u>pip install -r requirements.txt</u>

**STEP 5: Run the Website**<br>

<u>python app.py</u>

Open this link in your browser

<u>http://127.0.0.1:5000 </u>
